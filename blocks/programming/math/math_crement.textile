Incrementing/Decrementing a variable


h3. Operation

!help?f=blocks/programming/math/resources/math_crement_open_selection.png(math crement)!

* decrement
* increment

Decrementing a variable reduces the number by 1 (for instance if the variabl has a value of 10 initially, it will now have a value of 9) and incrementing a variable adds one to the variable. Because incrementing/decrementing a variable is a numeric operation, decrement/increment only works with integers and floats.
