h3. Pin number

!help?f=blocks/input-output/pin-states/resources/check-pin-input-drag-pin.png(check pin)!

* number

Each number indicates a different pin number. 0 will correspond with the pin labeled 0, 1 will correspond with the pin labeled 1, and so on. Use a math block or the output of another block to select a pin number.
